# LLM-Augmented Sensor Fusion for Urban Socioeconomic Monitoring

This repository provides a professional simulation framework for urban socioeconomic monitoring from a **Cyber-Physical-Social Systems (CPSS)** perspective. It replicates and validates the methodology proposed in the research article: *"LLM-Augmented Sensor Fusion for Urban Socioeconomic Monitoring: A Cyber-Physical-Social Systems Perspective"*.
